Welcome to eda(Event-Driven Architecture) components
=============================================================================== 

EDA(Event-Driven Architecture) components is an Event-Driven Architecture components which is not based on Spring.
It is a derivative from Spring Integration (http://www.springsource.org/spring-integration) which is based on Spring.
Spring Integration provides an extension of the Spring programming model to support the well-known Enterprise Integration Patterns
(http://eaipatterns.com/toc.html).
